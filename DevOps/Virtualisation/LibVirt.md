# LibVirt

## Overview

LibVirt is a collection interfaces to manage Virtual Machines. 
It is not a hypervisor but a standard way to interact with Hypervisors. 
Such as:
- KVM
- QEMU
- VirtualBox

## Installation
### Fedora 
On Fedora it is as easy as running:
`sudo dnf install @virtualization`
